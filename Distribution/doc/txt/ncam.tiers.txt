NAME
       ncam.tiers - TIER configuration file for NCAm

SYNOPSIS
       TIER mappings

DESCRIPTIONS
       CAID[,CAID]...:TIER ID|description

	  mapping between CAID, TIER ID and description of TIER

ANNONTATIONS
       Please use Unix text file format only.

EXAMPLES
	0001,0002,0003:000a|my TIER 1
	0004:000b|my TIER 2
	0005:000b|my TIER 3
