NAME

ncam.guess - CAID guessing table for NCAm  

SYNOPSIS

CAID guessing table  

DESCRIPTIONS

<length of ECM>:<CAID>

CAID guessing table by len in hex, only needed for protocols (at the moment BOMBA protocol only) that does not pass CAIDs
