NAME

ncam.srvid2 - service ID configuration file for NCAm  

SYNOPSIS

service ID mappings  

DESCRIPTIONS

service ID:CAID[:@provider ID[@provider ID]...][,:CAID[:@provider ID[@provider ID]...]][name]|[type]|[description]|[provider]

mapping between service ID, CAID, provider ID, name, type, description and proivder
 

ANNONTATIONS

Please use Unix text file format only.

You only need the ncam.srvid2 when using the monitor or the web interface. For saving memory consumption only insert the service IDs you really need. Some external programs use their own ncam.srvid2 and do not need the ncam.srvid2 of NCAm.
